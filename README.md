# Samoa Pickleball Hub

## Team Members
- Celestine Ahfua

## Project Overview
Samoa Pickleball Hub is a .NET Blazor web application designed to support the growing pickleball community in Samoa.

The application will help players and organizers:
- Create player profiles
- Track match results
- Schedule tournaments and community events
- Share announcements
- Display rankings and leaderboards

## Target Audience
- Pickleball players in Samoa
- Tournament organizers
- Local sports communities
- Community recreation groups

## Planned Features
- User/player profiles
- Tournament scheduling
- Match tracking
- Community announcements
- Leaderboard system
- Event management
