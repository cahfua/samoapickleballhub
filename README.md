# Samoa Pickleball Hub

## Team Members
- Celestine Ahfua

## Project Overview
Samoa Pickleball Hub is a .NET Blazor web application designed to support the growing pickleball community in Samoa. The application provides a centralized platform for players, tournaments, match results, announcements, and leaderboards.

## Target Audience
This application is intended for pickleball players, tournament organizers, and community members in Samoa.

## Features
- User authentication with register, login, and logout
- Player profile management
- Tournament scheduling
- Match result tracking
- Community announcements
- Leaderboard rankings
- Seeded sample data
- Responsive modern design
- Error handling and validation
- User guide documentation

## CRUD Functionality
The application includes create, read, update, and delete functionality for:
- Players
- Tournaments
- Match Results
- Announcements

## Technology Stack
- ASP.NET Core Blazor
- C#
- Entity Framework Core
- SQLite
- ASP.NET Core Identity
- HTML
- CSS
- Bootstrap
